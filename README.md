# JOINED-THESIS-ANNA-ALBERO-FORTE
This repository contains the full analytical pipeline developed for the study "How can data-driven methods improve operational planning in injection-moulding processes at PlasticForte through mold-change scheduling optimization?"
It develops and evaluates a two-stage framework that combines machine learning-based setup time prediction with Mixed-Integer Linear Programming (MILP) scheduling optimisation to improve mold-change scheduling at PlasticForte, a Spanish injection moulding manufacturer.
